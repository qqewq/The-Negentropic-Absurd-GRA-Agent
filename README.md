# The Negentropic Absurd GRA Agent

*From absurdity to order – iterative entropy reduction for verifiable innovation.*

## Core idea
The **Generative Recursive Absurdity (GRA)** agent starts with maximally absurd (high‑entropy) concepts and, through repeated cycles of constraint application, logical filtering, and empirical verification, transforms them into **practically applicable, low‑entropy structures** – new scientific theories, technological inventions, and art forms.

If absurdity is made **less entropic**, then absurd theories can be iteratively turned into increasingly negentropic, verifiable applications.

This repository provides:
- A rigorous information‑theoretic framework (entropy of absurdity, negentropy gradients)
- Mathematical derivations of convergence
- Documented, peer‑reviewable examples in **physics**, **engineering**, and **generative art**
- A Python implementation of the GRA agent loop
- Jupyter notebooks with reproducible experiments
